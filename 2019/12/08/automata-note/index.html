Automata课程笔记

class 1 自动机

  1. 自动机：识别正规文法所定义的语言和正规式所表示的集合 （课件中的language）
  2. 5元组 (Q,Σ,δ,q0,F) 有限状态集、输入集合、状态转移映射、开始状态、结束状态

class 2 DFA

  1. 构造自动机识别字符串：按字符串顺序自左向右逐个推导每个字符可能分支
  2. page 11：语言封闭性：L的补集也是正则语言（构造相反自动机）、已有正则语言字符串的数量组合也是正则语言，即正则表达式中使用的*号
  3. 闭包的正则运算规律

class 3 NFA

  1. NFA相对于DFA的变化：状态转移映射、初始状态为集合，且因为NFA的转换函数f为K*全集到K的子集的一种映射，所以NFA中可以有epsilon空转移
  2. NFA确定化：列表求解即可，从初始集合开始，输入定义的状态转移边获得新集合，对于每一个集合执行以上操作，直至不再出现新集合。

class4 闭包与正则表达式

  1. 闭包原理：并集、拼接、星号操作
  2. 正则表达式（表达正则语言的工具）常用符号 优先级：* > | > .

class 5 DFA限制

  1. DFA限制：状态数量固定
  2. Pumping lemma: 协助理解：文法类型
    chomsky文法类型：
    3型文法：正规文法：任一产生式α→β的形式都为A→aB或A→a
    2型文法（上下文无关文法）：对任一产生式α→β，都有α∈VN ， β∈(VN∪VT)*
    例 文法G[S]： S→AB A→BS|0 B→SA|1
    1型文法：1型文法产生式的一般形式是αAβ→αBβ， α,β ∈ V* ，A∈VN , β∈V＋ ，它表示当A的上文为α且下文为β时可把A替换成B，因此称1型文法为上下文有关文法
    0型文法(短语文法)：对任一产生式α→β，都有α∈(VN∪VT)且至少含有一个非终结符， β∈(VN∪VT)
  3. 正规语言可通过自动机的定义方式归约到正则形式 长度必然小于等于原匹配串 应用 判断是否可写成自动机形式
  4. 证明不是正规文法：把正则表达式写成确定形式, 构造u,v, 分别求出u,v,w, 保证 |z| >= N和 |uv| <= N，(N为非初始终止态的状态数量), 使v落入有限/方便求值的范围，选择一个i 和 k 推出构造的字符串∉L。从简单的字符串来看，构造正则表达式无法识别的特定数量的字符串形式。

class 7 PDA

  1. 状态转移除了输入外可指定栈操作，放宽了有限自动机从左至右逐个推导的形式，使用栈来记录已有状态，和上下文无关语法是等价的
  2. 小的 技巧：特殊字符和epsilon结合可知栈是否为空的信息；

class 8 CFG

  1. 开始状态S，中间状态大写字母，小写字母或特殊符号为常量
  2. L(M)和L(G)在另一者中的形式 TODO

class 9 PDA和CFG等价性 TODO

  1. high-level feature： 从CFG可构造PDA，循环结构可用来替换规则中最右边的值；只能处理当前字符串的值；自动机接受状态：栈和字符串全空
  2. 等价性：
    CFG到PDA：用栈模拟最左派生，对于文法中的生成式可用一个状态表示，栈进行状态替换；对于终结符，若读入字符串和栈顶匹配，则进行弹出（抵消）。最终使用空栈接收。
    从PDA到CFG：定义三元组形式，使用栈接收规则构造生成式

class 13 图灵机理论

  1. Recognition: 有限时间内能判断是否接受，但拒绝的情况可能要无限循环。Co-Turing-recognizable有限时间内能判断拒绝，但接收的情况可能要无限循环。decidability有限时间内进行接受或拒绝，为上两者的交集。三者区别。
  2. 消除左递归 from A→Aβ|γ transform to A→γA’, A’→βA’|ε
  3. TM acceptance undecidable不可判定：反证法：H says D accepts ⟨D⟩. But according to D, D rejects ⟨D⟩; H says D rejects ⟨D⟩. But according to D, D accepts ⟨D⟩.

附理发师悖论：他只为，而且一定要为，城里所有不为自己刮胡子的人刮胡子。如果他为自己刮胡子，那么按照他的豪言“只为城里所有不为自己刮胡子的人刮胡子”他不应该为自己刮胡子；但如果他不为自己刮胡子，同样按照他的豪言“一定要为城里所有不为自己刮胡子的人刮胡子”他又应该为自己刮胡子。 S={a|a is not in a}

  1. Decidability: TM halts on all inputs
    Recognizability: TM halts on YES cases but may not halt on NO cases
    Co-recognizability: TM halts on NO cases but may not halt on YES cases

总结

  1. 各类自动机的主要区别是它们能够使用的信息存储空间的差异：有限状态自动机只能用状态来存储信息；下推自动机除了可以用状态以外，还可以用下推存储器(栈)；线性带限自动机可以利用状态和输入/输出带本身。因为输入/输出带没有“先进后出”的限制，因此，其功能大于栈；而图灵机的存储空间没有任何限制。
  2. 自动机和文法的本质对应：NFA,DFA == 正规文法Regex；PDA == 上下文无关语法CFG；
    应用：编译原理
