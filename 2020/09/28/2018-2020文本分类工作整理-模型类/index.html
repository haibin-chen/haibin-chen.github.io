2018-2020文本分类工作整理(模型类)

不完全地整理一部分最近的文本分类模型的改进，对于motivation比较相近的文章（如用句法树建图做文本分类），只挑选部分代表性文章。

常见文本分类任务

传统文本分类：长度不同、领域不同（新闻、电影评论）

Aspect-based 情感分类：设计情感词与文本的交互（句法联系/两部分的特征融合方式）；复杂任务如同时抽取情感词和情感分类；

（层次化）多标签文本分类：理解任务要求（标签格式）的变化

2018-2020 NLP会议

传统文本分类

  1. Joint Embedding of Words and Labels for Text Classification 2018 ACL 引入标签信息
  2. Disconnected Recurrent Neural Networks for Text Categorization 2018 ACL 切割、分段RNN
  3. Semantic-Unit-Based Dilated Convolution for Multi-Label Text Classification 2018 EMNLP dilate结构+hybrid attention
  4. Heterogeneous Graph Attention Networks for Semi-supervised Short Text Classification 2019 EMNLP 短文本分类，建立了词 实体 主题的异构图

细粒度分类

  1. Syntax-Aware Aspect Level Sentiment Classification with Graph Attention Networks 2019 EMNLP 图注意力网络（句法树建图）
  2. Aspect Sentiment Classification with Document-level Sentiment Preference Modeling 2020 ACL 同一类aspect和同一文档aspect，建图

多标签分类

  1. Label-Specific Document Representation for Multi-Label Text Classification 2019 EMNLP 标签空间+文本空间embedding的动态融合
  2. SGM: Sequence Generation Model for Multi-label Classification 2018 COLING 序列生成做多标签文本分类，引入标签间联系
  3. Ranking-Based Autoencoder for Extreme Multi-label Classification 2019 NAACL 引入基于margin loss的自编码器，文本和标签特征融合后可重构标签

层次化多标签

  1. HFT-CNN: Learning Hierarchical Category Structure for Multi-label Short Text Categorization 2018 EMNLP 层数加权loss
  2. Hierarchy-Aware Global Model for Hierarchical Text Classification 2020 ACL 图网络建模标签层次
  3. Hierarchical Text Classification with Reinforced Label Assignment 2020 ACL 使用强化学习建模马尔科夫搜索路径，在层次标签树中进行搜索

可解释性类

  1. Rethinking Complex Neural Network Architectures forDocument Classification 2019 NAACL
    在合适的正则化策略下，一个LSTM(无attention)在文本分类下也能达到较好效果

  2. Text Classification with Few Examples using Controlled Generalization 2019 NAACL
    从大型外部语料任务中提取针对性有效（合适的范围、泛化特征）特征如ngram提取器、共现性提取器，用于少数据的对应任务中。

  3. Generating Hierarchical Explanations on Text Classification via Feature Interaction Detection 2020 ACL 特征交互来分割和分层、评估词对于模型预测结果的贡献度

    其他

    1. Zero-shot Text Classification via Reinforced Self-training 2020 ACL 零样本文本分类，强化学习学习自学习策略，即在测试类别挑出置信度高的做为伪标签
    2. Out-of-Domain Detection for Low-Resource Text Classification Tasks 2019 EMNLP
      训练一个few shot网络做二分类的OOD异常检测任务
    3. Deep Unknown Intent Detection with Margin Loss 2019 ACL 借鉴OOD异常检测任务中的密度检测，训练识别新意图

2018-2020其余顶会如NIPS ICML KDD

传统文本分类

  1. DiSAN: Directional Self-Attention Network for RNN/CNN-Free Language Understanding 2018 AAAI 有向注意力机制，以及相关的方向注意力机制工作
  2. Multi-Scale Self-Attention for Text Classification 2020 AAAI transformer引入多尺度结构
  3. Ordered Neurons: Integrating Tree Structures Into Recurrent Neural Networks 2019 ICLR LSTM或CNN结构研究范例，层次化的有序LSTM
  4. Tensor Graph Convolutional Networks for Text Classification AAAI 2020 设计了语义、句法和序列图及其交互

多标签文本分类

  1. AttentionXML: Extreme Multi-Label Text Classification with Multi-Label Attention Based Recurrent Neural Networks 2019 NIPS K-means构造元层次化标签和标签树
  2. Two-Stage Label Embedding via Neural Factorization Machine for Multi-Label Classification 2019 AAAI 用FM建立label表示的联系，loss考虑了标签相关性
  3. Taming Pretrained Transformers for Extreme Multi-label Text Classification KDD 2020 先聚类 分类器选出最大的类别后进行ranking排序，避免了极多标签的搜索问题

总结

  1. 这两年多了很多基于transformer减少参数、适应长文本等的研究，当然也对应着bert结构的蒸馏、压缩等探索
  2. 针对传统文本分类的变少了，增加了很多做细粒度情感分类的工作。今年有一些细粒度情感工作比较类似，都是围绕句法树、图神经网络等套上去，当然这一块也有一些新任务吧（比如联合识别情感词和情感类型）
  3. 多标签文本分类的工作也增加了，其中包括普通多标签文本、层次化多标签、极多标签（KDD NIPS）
  4. 元学习、小样本分类刚刚起步，基本都在原数据集上调整为few shot形式，暂时没有统一的评测或技术方向的标准。
  5. 总结一下图网络建图的几大形式：句法图、知识图谱关系建图、实体句子主题文档等多层次建图、多个句子之间建图、多对象等迭代更新（如一问一答的模式）….图网络过去一年把文本分类、实体识别、阅读理解、多轮QA等套了个遍，还有没有新玩法？
