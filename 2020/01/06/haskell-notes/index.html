map取一个函数和List做参数，遍历该List的每个元素来调用该函数产生一个新的List。
map (+3) [1,5,3,1,6]与[x+3 | x <- [1,5,3,1,6] 等价
filter函数取一个限制条件和一个List，返回该List中所有符合该条件的元素。
flip，它的作用是将两个参数的顺序互换。
zip允许在2个列表中，每次抽取列表中的一个元素，进行组合操作：

语言特性：

  1. 类型检查：:t
    类型定义行为的接口 e.g.(==) :: (Eq a) => a -> a -> Bool : 相等函数取两个相同类型的值作为参数并返回一个布尔值，而这两个参数的类型同在Eq类之中（即类型约束）

  2. 模式匹配：按模式从中取得代码命令 _为通用匹配符号
    e.g. List Patterns [1,2,3]本质就是1:2:3:[]的语法糖，x:xs这样的模式可以将list的头部绑定为x，尾部绑定为xs。如果这list只有一个元素，那么xs就是一个空list。
    Lambda expression 更加简洁！

  3. higher order 应用：filter
    fold取一个二元函数 包含一个初始值和折叠(fold)list的最后一个值
    foldr(:) [ ] xs : (:)类似于+的操作符, []为list comprehension迭代中的空list 即fold的开始

PPT page 22:It also makes obvious the fact that foldr (:) [] is the identity function on lists, as replacing cons with cons and nil with nil will not change anything.
$: 左结合转换为右结合

Quiz : implement Data.Set

必須具有 Ord 的功能！ 也可以是插入时做判断: 

insert :: Integer -> Set Integer -> Set Integer
insert x (Set xs)
| not (x `elem` xs) = Set (x:xs)
| otherwise         = Set xs

一些idea: 构造set (filter (/= x) xs) 交集 filter ('elem' xs2) xs1 并集 filter ('repeat' xs2) xs1 ++ xs2
